# 📚 StudyBuddy AI

An AI-powered PDF study assistant that helps students understand and revise their study material through intelligent Q&A.

---

## 🚀 Features

- 📄 Upload any PDF (textbook, notes, question papers)
- 💬 Ask questions directly based on the uploaded document
- 🤖 Powered by **Groq's LLaMA 3.3 70B** model for fast, accurate answers
- 🧠 Extracts and understands content using **PyPDF2**
- ⚡ Built with **Streamlit** for a clean, interactive UI

---

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| Python | Core language |
| Streamlit | Web UI |
| PyPDF2 | PDF text extraction |
| Groq API (LLaMA 3.3 70B) | AI question answering |

---

## 📦 Installation

1. **Clone the repository**
```bash
git clone https://github.com/Pruthvi-H123/studybuddy-ai.git
cd studybuddy-ai
```

2. **Install dependencies**
```bash
pip install -r requirements.txt
```

3. **Set up your API key**

Create a `.env` file in the root directory:
```
GROQ_API_KEY=your_groq_api_key_here
```

4. **Run the app**
```bash
streamlit run app.py
```

---

## 💡 How to Use

1. Open the app in your browser
2. Upload a PDF file (notes, textbook chapter, etc.)
3. Type your question in the chat box
4. Get instant AI-powered answers based on your document!

---

## 📸 Use Cases

- Revising for exams using your own notes
- Understanding complex topics from textbooks
- Quick Q&A from research papers or assignments
